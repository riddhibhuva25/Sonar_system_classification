# 🚀 Sonar Signal Classification (Machine Learning Project)

## 📌 Project Overview

This project focuses on classifying sonar signals to determine whether the object is a **rock** or a **mine** using Machine Learning techniques.

The dataset contains sonar signals bounced off different surfaces, and the goal is to build a model that can accurately predict the type of object.

---

## 🎯 Objective

* Classify objects as **Mine (M)** or **Rock (R)**
* Apply Machine Learning algorithms for prediction
* Evaluate model performance using standard metrics

---

## 📊 Dataset Information

* Dataset: Sonar Dataset
* Total Instances: 208
* Features: 60 numerical attributes
* Target Variable:

  * `M` → Mine
  * `R` → Rock

---

## ⚙️ Technologies Used

* Python 🐍
* NumPy
* Pandas
* Scikit-learn

---

## 🧠 Machine Learning Workflow

### 1. Data Preprocessing

* Handling missing values
* Feature scaling using StandardScaler
* Train-test split

### 2. Model Training

* Logistic Regression model used

### 3. Model Evaluation

* Accuracy score

* Performance analysis
